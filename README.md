# SCHELPDESK

**Smart Campus Helpdesk System(UNIASSIST)** – A full-stack web application for managing student support tickets, FAQs, and a digital student handbook. It features intelligent ticket classification, a smart FAQ chatbot, and administrative dashboards for staff and faculty.

---

## Tech Stack
- **Frontend:** React.js, Vite  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Auth:** JWT authentication  
- **HTTP client:** Axios
